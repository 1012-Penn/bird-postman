# 开发更新记录

本文件记录项目面向试玩版本的重要变化。

## 2026-06-21 — 初始 Demo

- 建立单页可运行的 Windows 浏览器游戏原型。
- 加入山坡滑翔、风圈收集、分数与连击系统。
- 加入轻松的背景视觉、粒子效果和基础 Web Audio 音效。
- 将主角由风筝改为圆头圆脑的小鸟。
- 调整物理：按住会贴紧地面蓄力，释放会产生明显的借坡起飞效果。

## 2026-06-21 — 斜坡滑翔物理重构

- 地形改为可无限查询的 Catmull-Rom spline 山丘；每个位置均提供高度、切线、法线、坡度角和曲率。
- 加入基于切线重力分量的 Grounded 状态，以及包含重力、俯冲与滑翔受力的 Airborne 状态。
- 加入基于速度投影与入射方向的落地质量计算；顺坡落地会保留更多速度并得到奖励。
- 加入基于地形曲率和法向约束的自然起飞，不再永久锁定在地面。
- 新增默认开启的调试层：速度向量、切线、法线、状态、速度、坡度与落地质量；按 `D` 切换。

## 2026-06-21 — 节奏与速度调整

- 显著提高俯冲时的坡面重力加速度，并降低地面摩擦和空气阻力，使多次成功俯冲的速度能连续累积。
- 提高地形振幅并缩短波长，形成更陡、更紧凑的连续山丘。
- 起点自动选择在下坡段，并携带可立即进入节奏的初速度。
