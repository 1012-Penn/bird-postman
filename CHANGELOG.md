# 开发更新记录

本文件记录项目面向试玩版本的重要变化。

## 2026-06-21 — 初始 Demo

- 建立单页可运行的 Windows 浏览器游戏原型。
- 加入山坡滑翔、风圈收集、分数与连击系统。
- 加入轻松的背景视觉、粒子效果和基础 Web Audio 音效。
- 将主角由风筝改为圆头圆脑的小鸟。
- 调整物理：按住会贴紧地面蓄力，释放会产生明显的借坡起飞效果。

## 2026-06-21 — 斜坡滑翔物理重构

- 地形改为可无限查询的样条山丘；每个位置均提供高度、切线、法线、坡度角和曲率。
- 加入基于切线重力分量的 Grounded 状态，以及包含重力、俯冲与滑翔受力的 Airborne 状态。
- 加入基于速度投影与入射方向的落地质量计算；顺坡落地会保留更多速度并得到奖励。
- 加入基于地形曲率和法向约束的自然起飞，不再永久锁定在地面。
- 新增默认开启的调试层：速度向量、切线、法线、状态、速度、坡度与落地质量；按 `D` 切换。

## 2026-06-21 — 节奏与速度调整

- 显著提高俯冲时的坡面重力加速度，并降低地面摩擦和空气阻力，使多次成功俯冲的速度能连续累积。
- 提高地形振幅并缩短波长，形成更陡、更紧凑的连续山丘。
- 起点自动选择在下坡段，并携带可立即进入节奏的初速度。

## 2026-06-21 — 非对称山丘

- 地形改为连续的三次 Bézier 山丘：下坡占一个周期的 72%，上坡缩短至 28%。
- 这让俯冲能在长下坡持续累积动能，并将短上坡作为速度不足时的惩罚与恢复段。

## 2026-06-21 — 柔和山丘与加速调整

- 地形改为正弦函数，保证高度、切线与曲率完全连续，并让上、下坡长度恢复对称。
- 俯冲时的地面与空中受力提高 50%，以便一个完整下坡能建立可用于越过下一座山脊的速度。
- 上坡时的重力阻力降低 30%，增强速度不足时的自主爬坡能力。

## 2026-06-21 — 飞行反馈

- 新增右上角的一键重玩按钮，可随时重置本轮飞行。
- 山丘振幅减少 50%，重力减少 30%，以提供更高、更长的滞空时间。
- 小鸟离开坡面持续飞行满一秒时，会随机发出 5 种可爱的拟人化叫声之一。

## 2026-06-21 — 强化俯冲

- 贴地俯冲的坡面加速再提高 50%，空中俯冲下压力同步提高 50%。

## 2026-06-21 — 动量保留

- 小鸟离坡时会保存当前动量；后续落地不会损失已累积的速度，顺坡落地还可继续获得增益。
- 零速度时的最低爬坡速度提高 50%（25 → 37.5 px/s）。

## 2026-06-21 — 高速坡顶起飞

- 增加坡顶前瞻检测：高速接近坡顶时会主动离开坡面，保留上坡速度并获得小幅抬升。
- 保留原有基于曲率与法向约束的自然起飞判定。
