# 风筝邮差 / Bird Postman

一个可在 Windows 浏览器中运行的单机滑翔小游戏原型。

## 运行

直接打开 `index.html`，或在项目目录启动本地服务器：

```powershell
python -m http.server 8080
```

然后访问 `http://127.0.0.1:8080`。

## 操作

- 按住空格、鼠标或触摸：小鸟贴地蓄力。
- 松开：将蓄力转化为起飞速度。

## 当前功能

- 起伏地形与滑翔物理
- 贴地蓄力与释放起飞
- 风圈、里程和连击
- 圆润小鸟角色、动态天空、基础音效
